# bright-coffee-shop-sales-dashboard
This project presents an interactive sales dashboard for Bright Coffee Shop, developed using both Power BI and Google Looker Studio (Data Studio). The dashboard provides a comprehensive analysis of sales performance, revenue trends, product categories, and store locations to support data-driven decision-making.

# Objectives
-Analyze overall revenue performance
-Monitor sales trends over time
-Compare revenue across store locations
-Identify top-performing product categories
-Provide interactive filtering for deeper analysis

# Tools Used
-Power BI
-Google Looker Studio (Data Studio)
-Microsoft Excel / CSV Dataset

# Key Performance Indicators (KPIs)
-Total Revenue
-Total Transactions
-Unique Products Sold
-Store Locations

# Dashboard Features
-Revenue by Product Category
-Revenue by Store Location
-Revenue Trend Analysis
-Revenue by Day of the Week
-Interactive Filters and Slicers
-Detailed Sales Summary Table

# Insights Generated
-Coffee products generated the highest revenue
-Revenue distribution varied across store locations
-Weekday sales contributed more revenue than weekends
-Certain product categories consistently outperformed others

#Project Deliverables
-Power BI Dashboard 
-Looker Studio Dashboard
-Dataset
